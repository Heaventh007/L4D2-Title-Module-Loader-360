# L4D2-Title-Module-Loader-360
This loader takes advantage of L4D2 loading a dll, and loads our own instead.

# How to use
- Compile the loader, and take the `vxbdm_360.dll` file and place it in the bin folder next to the other ones.
- Put all of your patches, hooks and whatever else, in the concept dll project.
- Place the HPatch.dll file next to the default.xex file.
